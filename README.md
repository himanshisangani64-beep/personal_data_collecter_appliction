<h1>Personal Data Collector Application (Python)</h1>

<h2>Project Description</h2>

<p>This project is an Interactive Personal Data Collector Application in which personal information is collected from the user, processed, and then displayed neatly.</p>

<h2>Objectives</h2>

<ul>
<li>Collect data from the user</li>
<li>Process the collected data</li>
<li>Display the results</li>
<li>Demonstrate Python fundamentals</li>
</ul>

<h2>Assumptions</h2>

<ul>
<li>The user will provide valid input</li>
<li>Age will be entered in numeric format</li>
<li>Height will be entered in meters</li>
</ul>
